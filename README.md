# ⚙️ Cloud Resume Project (Backend)

---

## 📌 Overview

This repository contains the backend of the Cloud Resume application, built using AWS serverless services.

It provides a REST API to track and update visitor counts.

---

## 🏗️ Architecture

---

## ⚙️ Tech Stack

- AWS Lambda (Python)
- AWS API Gateway (HTTP API)
- AWS DynamoDB
- Terraform (Infrastructure as Code)

---

## 📂 Structure

## ⚙️ How It Works

1. API Gateway receives request  
2. Lambda function executes (Python)  
3. DynamoDB updates visitor count  
4. Response returned to frontend  

---

## ✨ Features

- Serverless backend architecture
- Visitor counter API
- Fully automated deployment via Terraform
- CI/CD pipeline with GitHub Actions
